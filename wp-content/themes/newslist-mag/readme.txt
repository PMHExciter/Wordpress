== Description ==
Newslist Mag is a fast, clean, modern-looking Magazine WordPress theme. It is SEO friendly, Gutenberg support, translation and page builder ready. Newslist mag is fit for dynamic news, newspapers, magazine, publishers, blogings, editors, online and tech magazines and creative business. Check out Live preview : https://preview.risethemes.com/sites/newslist-magazine/

For more information about Newslist Mag please go to https://risethemes.com/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
